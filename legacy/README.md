# legacy/ — Loop Playground v1 (reference only)

`v1/index.html` + `v1/app.js` are the **original static Loop Playground**, preserved as a
reference while v2 is built. They are **not** part of the v2 build or workspace.

Per `docs/V1-AUDIT.md`, v1's *teaching soul* is carried forward into v2 (the four-beats
visual language, the two-exits/runaway lesson, the checkability linter, copy-as-prompt,
the honest "Simulation" tone) while its **fake, hard-coded engine** is replaced by the real
`@loop-lab/loop-core` runner and data-driven lessons.

To view v1: open `legacy/v1/index.html` in a browser. Do not add features here — build them
in `apps/` and `packages/`.
